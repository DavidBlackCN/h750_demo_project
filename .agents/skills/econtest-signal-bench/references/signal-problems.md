# 信号题选参

## 基本波

- SINE：适合增益、相位、滤波和频率响应。
- SQUARE：适合边沿、带宽、过冲和数字阈值观察。
- RAMP：适合线性、积分、比较器和扫描类观察。
- PULSE：明确设置频率、宽度、占空比和边沿时间。

## 示波器起始设置

- 垂直档位可从预期 Vpp 的约 1/5 每格开始。
- 时基可从周期的约 1/5 每格开始，使屏幕包含约两个周期。
- 触发源选择实际观察通道，电平从波形中点开始。
- 先用 AUTO 找到稳定信号，再根据任务选择 NORMAL 或 SINGLE。

## 扫频

扫频只支持 SINE、SQUARE、RAMP。记录起止频率、时间、LINE/LOG 和方向。软件依次操作两台 USB 设备，不得声称硬件同步。

## 有效性

自动测量可能因短窗口或非平稳扫频返回无效值。此时检查原始波形覆盖时间、Vpp、周期数量和占空比；证据不足则标记 inconclusive。
