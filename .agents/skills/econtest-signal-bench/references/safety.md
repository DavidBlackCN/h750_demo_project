# 安全边界

## 人工负责的物理条件

软件不能验证探头实际位置、保护接地、隔离、额定值、BNC 外壳电位或 DUT 上电状态。访问硬件前，要求操作者确认这些条件。

示波器和台式信号源通常保护接地。不得将示波器地夹或 BNC 外壳连接到非隔离高侧节点。市电、大功率、电机驱动和非隔离电源不属于本项目范围。

## 信号源

- 输出 ON 前确认通道、目标、负载语义和最大 3 Vpp 限制。
- High-Z 接收端使用 `HZ`；只有真实 50 Ω 端接时才使用 `50`。
- 配置基本波、负载或扫频前，输出必须回读 OFF。
- offset 首版固定为 0 V。
- OFF 不需要危险动作确认，但必须查询确认；查询失败时状态为 unknown。

## 示波器

- 探头倍率必须与实物一致。
- RUN、STOP、SINGLE 会改变前面板状态，需要明确授权。
- 有界波形读取默认保持 STOP。
- AUTO 尚未封装为公开工具。

## 会话授权

线路、负载、电压、隔离、目标通道或任务范围改变后，旧确认失效。布尔确认只是防误触门，不是身份认证，也不能替代物理安全判断。
